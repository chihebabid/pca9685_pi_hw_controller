Repository overview
-------------------
This repository contains two related ROS2 packages in the same folder:

- `pca9685_pi_hw_interface` — the hardware interface package that implements the ros2_control `SystemInterface` for the PCA9685. See `src/pca9685_pi_hw_interface/README.md` for detailed documentation.
- `demos_pca9685_hw_interface` — a demo package that supplies a URDF/xacro, `controllers.yaml`, and a launch file to run a 3-servo example. See `src/demos_pca9685_hw_interface/README.md` for usage and quickstart.

Open the per-package README files for more details.
